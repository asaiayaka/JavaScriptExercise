# 解答

## localStorage

- 確認方法
  1. ToDoを追加してlocalStorageに保存
  2. ページをリロード
  3. ブラウザを完全に閉じる
  4. 再度ブラウザを開いてページを表示

- 結果
   - ページをリロードしてもデータは残った
   - ブラウザを閉じて再度開いても残った

- 結論
  開発者・ユーザーが消さない限り残る

## sessionStorage

- 確認方法
  1. ToDoを追加してsessionStorageに保存
  2. ページをリロード
  3. タブを完全に閉じる
  4. 同じURLを別タブで開く
  5. ブラウザを閉じて再度開く

- 結果
   - ページをリロードしてもデータは残った
   - 同じタブにいる限りデータは維持される
   - タブを閉じるとデータが消える
   - ブラウザを閉じても消える
   - 別タブで同じURLを開いてもデータは共有されない

- 結論
  sessionStorage のデータの有効期限はタブを閉じるまで
