# 解答

## package-lock.jsonの役割

package-lock.jsonは、実際にインストールされた依存関係の完全なスナップショットを記録するファイル

記載される内容：

- 各パッケージのバージョン
- 依存関係の依存関係(再帰的)
- 取得元(URL)
- 整合性チェックハッシュ

必要性：

package.jsonだけでは^1.2.3のようにバージョン範囲指定しかわからないため、package-lock.jsonがあることでどの環境でも同一の依存関係ツリーを再現可能。

## リポジトリにコミットすべきか

原則すべき。
再現性の確保・CI / テスト環境の安定・チーム開発での依存関係のずれ防止のため

例外：npmパッケージライブラリ開発の場合
