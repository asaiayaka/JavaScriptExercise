# 解答

npxを利用する利点は、ローカルにインストールされたnomパッケージを、グローバルインストールせずに直接実行できる点。

詳細：

1. グローバルインストールが不要
   - eslint, webpack, jestなどのCLIツールをnpm install -gせずに使うことができる
   - 環境差異やバージョン不一致を防げる
2. プロジェクトごとのバージョンを保証できる
   - node_modules/ .binにあるそのプロジェクト専用のバージョンを実行
   - チーム全員が同じツールで作業できる
3. 一時的なツールの実行が容易
   - インストールせずに一度だけ実行可能
4. npm scriptsとの親和性
   - scripts内ではnpx相当の解決が自動で行われる
